# Changelog

Все заметные изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2025-12-07

### Добавлено
- Полнофункциональный менеджер медиа-файлов
- Загрузка файлов (изображения, видео, документы)
- Иерархическая структура папок
- Корзина с возможностью восстановления
- Фильтрация и поиск файлов
- Пагинация и сортировка
- RESTful API для работы с медиа
- Vue компонент для фронтенда
- Мультипользовательская поддержка (scope по user_id)
- Защищенные папки
- Drag & Drop для папок
- Мягкое удаление
- Автоматическая генерация превью для изображений

### Технические детали
- PHP >= 8.2
- Laravel >= 10.0
- Поддержка Laravel Sanctum для авторизации
- PSR-4 autoloading
- Конфигурируемые настройки через config файл
- Миграции для создания таблиц
- Service Provider для автоматической регистрации

### Документация
- Полная документация API
- Инструкция по установке
- Примеры использования
- Конфигурация и настройка
